MaxSoft API Health Checker

Project Information

  • Category: Real time API health check dashboard
  • Description: A real time health checking dashboard to monitor the availability of all the backend services integrated with any project in any environment
  • Advantages: Easy to find unavailable backend services, Real-time updating the services status in every "x" seconds, Easy to integrate services, No need to worry about the CORS issue, Both Web and Mobile friendlyUI, Free and open-source
  • Technologies: NodeJS, Express, JavaScript, HTML and CSS
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: JavaScript
  • Project Date: 7 Nov, 2020
  • Publication URL: https://medium.com/api-health-checker
  • Project URL: https://github.com/osandadeshan/api-health-checker-dashboard
