MaxSoft GReporter

Project Information

  • Category: Email reporter plugin for Gauge framework
  • Description: A plugin for Gauge framework to send the test execution summary report with graphical representation to a defined audience via an email
  • Advantages: Automated emails for test execution summary with graphical representations, Email templates can be configurable, Ability to define the receipient list in environment wise, Ability to use any email address as the sender email
  • Technologies: Java, Gauge Framework, JFreeChart, xChart, SMTP, MailAPI and Maven
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: Java
  • Project Date: 27 Jun, 2018
  • Publication URL: https://medium.com/greporter
  • Project URL: https://github.com/osandadeshan/MaxSoft-GReporter
