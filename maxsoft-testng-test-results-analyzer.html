MaxSoft   TestNG Test Results Analyzer

Project Information

  • Category: Reporter plugin for TestNG to analyze the failed and skipped tests easily in test automation
  • Description: An easily integratable reporting plugin which automatically generates an Excel file with the failed and skipped tests against the reason. On the same Excel file, the second and third tabs contain the failure and skipped reasons grouping. So it would be easy to identify the tests which failed or skipped due to the same reason. Further more, this also generates the Extent report after the execution
  • Advantages: Automatically generates the Test Analysis Report after the test execution, Automatically generates the Extent Report after the test execution, Reporter details can be configured through a property file, No need to implement classes for Extent reporter or Test Analysis reporter, Simple and easy to use
  • Technologies: Java, TestNG, Apache POI, Extent Report, Selenium and Maven
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: Java
  • Supported Test Runners: TestNG
