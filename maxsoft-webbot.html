MaxSoft WebBot

Project Information

  • Category: Codeless Web UI test automation framework
  • Description: A simple, code-free and efficient framework for technical QE, non-technical QE or developer to perform Web UI automation in an easy manner
  • Advantages: Code free web UI automation, Human readable tests in business language and Markdown syntax, Tests can be designed even by a non-technical person, Generate a HTML report with test details for every test execution, Generation of an executable document, Parallel execution, Live execution report, Automated emails for test execution summary with graphical representations
  • Technologies: Java, Gauge Framework, Selenium, Apache POI and Maven
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: Java
  • Project Date: 7 May, 2018
  • Project URL: https://github.com/osandadeshan/MaxSoft-WebBot
