Text Extractor (OCR Scanner)

Project Information

  • Category: Application to convert image to text
  • Description: An android application which can read the text inside an image and output it as a text which you can share with your colleagues from your mobile phone
  • Advantages: Able to convert image to text easily, Able to manipulate the converted text, Able to share the text
  • Technologies: Java, Android Studio, C++, OCR
  • Supported Platforms: Android
  • Project Date: 21 Oct, 2016
  • Google Playstore URL: https://play.google.com/store/apps/details?id=com.osanda.ocr
  • Project URL: https://github.com/osandadeshan/Text-Extractor
