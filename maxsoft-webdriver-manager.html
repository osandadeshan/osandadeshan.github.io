MaxSoft WebDriver Manager

Project Information

  • Category: Plugin to spinup any automated browser instance
  • Description: A plugin to provide an easy way for quality engineers or automation engineers to spin-up any browser instance in any platform without any configurations
  • Advantages: No need of any configurations, All browsers are embedded, Easy to use, Platform independent
  • Technologies: Java, Selenium, Bonigarcia WebDriverManager and Maven
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: Java
  • Project Date: 15 Mar, 2020
  • Publication URL: https://medium.com/webdrivermanager
  • Project URL: https://github.com/osandadeshan/maxsoft-webdriver-manager
