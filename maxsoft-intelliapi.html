MaxSoft IntelliAPI

Project Information

  • Category: Codeless API test automation framework
  • Description: A simple, code-free and efficient framework for technical QE, non-technical QE or developer to perform API automation in an easy manner
  • Advantages: Code free API automation, Ability to validate backend databases,Human readable tests in business language and Markdown syntax, Tests can be designed even by a non-technical person, Generate a HTML report with test details for every test execution, Generation of an executable document, Parallel execution, Live execution report, Automated emails for test execution summary with graphical representations
  • Technologies: Java, Gauge Framework, Rest Assured, Apache POI and Maven
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: Java
  • Project Date: 27 May, 2017
  • Publication URL: https://medium.com/intelliapi
  • Project URL: https://github.com/osandadeshan/MaxSoft-IntelliAPI
