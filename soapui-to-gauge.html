SoapUI to Gauge Plugin (API Test Execution and Report Generation Plugin)

Project Information

  • Category: Reporter plugin for SoapUI
  • Description: A Java plugin to execute SoapUI test suite XML from Gauge framework and generate a detailed HTML report with all SoapUI test steps
  • Advantages: Enables execution report for the SoapUI free version, Easy to use, Free and open source
  • Technologies: SoapUI, Java, Gauge Framework, TestNG and Maven
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: Java
  • Project Date: 20 March, 2017
  • Project URL: https://github.com/osandadeshan/SoapUI-To-Gauge-Plugin
