MaxSoft Extent Reporter

Project Information

  • Category: Reporter plugin for test automation frameworks and projects
  • Description: An easily integratable reporting plugin which automatically generate the Extent report for any kind of TestNG based test automation framework
  • Advantages: Automatically generates the Extent report after the test execution, Reporter details can be configured through a property file, No need to implement classes for Extent reporting, Easy to use
  • Technologies: Java, Extent Report, Selenium and Maven
  • Supported Platforms: Windows, Linux and Mac OS
  • Supported Languages: Java
  • Project Date: 28 Nov, 2020
  • Publication URL: https://medium.com/extent-reporter
  • Project URL: https://github.com/osandadeshan/maxsoft-extent-reporter
